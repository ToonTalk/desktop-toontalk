Vi rekommenderar att du installerar ToonTalk innan du kör programmet. Några problem du kan få är bl a:
  1. Marty blir tyst. Den röst som är avsedd för Marty kan inte användas utan installation. På vissa datorer (sådana som har Microsoft SAPI 4.0 installerat och röster för text-till-tal) kan Marty ändå tala när en röst valts i "Välj Alternativ".
  2. I vissa versioner av Windows 95 och några versioner av Windows 98 kan det hända att ToonTalk inte går att köra alls utan installation. Var snäll och rapportera eventuella problem att köra med Windows Me, Windows 2000, Windows XP, eller senare.
