Musproblem i ToonTalk

När ToonTalk försöker lista ut hur din dators mus har rört sig, kan programmet få in omöjliga sifferdata. Detta har observerats när ToonTalk körs med ett emuleringsprogram som SoftWindowsä eller VirtualPCä på Macintoshä datorer.

Du kan köra ToonTalk utan att röra musen. Piltangenterna på tangentbordet kan användas i stället.

Om ett problem kvarstår, var vänlig rapportera till support@toontalk.com.

 

Om du är uppkopplad till Internet kan du klicka här för att få uppdaterad information om detta.
