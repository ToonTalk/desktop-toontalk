Att installera ToonTalk-tillägg

Enbart för avancerade användare:

ToonTalk-tillägg installeras inte som standard. En anledning till detta är att de gör det möjligt för robotar att ändra på saker utanför ToonTalks säkra värld. Arkiv-tillägget t ex, tillåter ToonTalks robotar att läsa och skriva (eller radera) vilken fil som helst i din dator. Det här är en styrka som kan användas både till gott och till ont. Om du installerar detta tillägg, var då säker på att du kan lita på de människor som ger dig ToonTalk-program att köra på din dator.

An extension is installed if the DLL file starting with "TT" followed by the extension name is in your ToonTalk directory. For example, the file extension file is "TTFile.dll".

Om du gör installationen från en CDROM, välj då "Anpassad installation". Du ska ta bort alla kryss utom de som står vid beteckningar på de tillägg du vill ha.

If you are installing from a CDROM, then choose the "Custom Installation" option. Select "Program Files" and click on the "change" button. You should then remove check marks from every component except the ones that are labeled with the extensions you desire. Alternatively copy the appropriate files from the \toontalk\extend subdirectory.

Om du gör installationen från Internet, ladda då ner ttfile.dll och kör den för att installera Arkiv-tillägget.
