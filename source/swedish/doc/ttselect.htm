Vad du kan göra om Marty inte hörs i ToonTalk

Om du har installerat ToonTalk från en CDROM och kör Windows 95, Windows 98, eller Windows NT 4.0, då bör Marty kunna tala. Här är några orsaker till att det ändå kan gå snett och tips på vad du kan göra åt dem:

  1. Ljudkort saknas. Inte så konstigt om Marty inte kan tala när operativsystemet inte kan hitta ljudkortet.
  2. Dåliga drivrutiner för ljud. Operativsystemet behöver ha program som kallas drivrutiner för att utnyttja ljudkortet. Du kanske inte har några och det kan vara fel eller för gammal version. Ta kontakt med den affär som sålt din dator eller ljudkortet för att få de senaste drivrutinerna.
  3. Annat. Som med all mjukvara finns det annat som kan gå snett. Du kanske vill köra ToonTalk utan tal genom att välja "Välj Alternativ" när du startat ToonTalk. Eller så kan du skriva och berätta om problemet.
