Att köra Microsoft Window's Media Control Interface

Microsoft Windows stödjer ett antal kommandon för uppspelning av ljud, musik, video och andra mediatyper. Du kan skicka dessa kommandon till Windows från MCI genom att hålla ett textkort som innehåller ett kommando och sen klicka på en musknapp medan du håller kontrolltangenten nedtryckt (brukar heta Ctrl). Tex om textkortet innehåller texten "play canyon.mid" och midifilen canyon finns i din Windowskatalog kommer du börja höra musiken. Om du sen klickar medan du håller ett textkort som innehåller texten = "close all" kommer musiken att sluta höras. Det finns flera böcker som beskriver Microsoft's MCI kommandospråk.
