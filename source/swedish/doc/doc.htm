ToonTalk Hjälpreda

När du klickar på ToonTalk ikonen , får du dessa valmöjligheter:

  1. Göra uppdrag. Det här är ett roligt sätt att lära sig hur ToonTalk fungerar och hur man kan göra läckra saker.
  2. Se demos. Du kan luta dig tillbaka och se på en introduktionsdemo med uppläst berättartext eller se hur 2 barn försöker bygga ett bordtennisspel eller titta på en massa andra demoprogram.
  3. Leka fritt. Det börjar med att du flyger över ToonTalk-ön i din helikopter. Marty kommer och hjälper dig så fort du går in i ett hus.
  4. Inställningar. Låter dig ställa in ToonTalk så det pasar din dator.
  5. Hjälpreda. Startar din webbläsare så du kan se denna fil.

Vill du veta mera om det som kommer här nere ska du bara klicka på de understrukna orden.

  • Att umgås med Marty.
  • Att använda Dammis, Pumpas och Trollstaven.
  • Sifferkort och textkort.
  • Lådor.
  • Fåglar och fågelbon.
  • Vågar för att jämföra saker.
  • Att lära upp robotar.
  • Lastbilar för att bygga nya hus.
  • Bomber för att ta bort hus.
  • Din anteckningsbok.
  • Att använda bilder och animationer.
  • Att använda ljudeffekter.
  • Att flyga och landa med helikoptern.
  • Att sitta ner och ställa sig upp.
  • Att göra saker på golvet.
  • Att ställa in ToonTalk.
  • Att använda sensorer.
  • Att använda ToonTalk tillägg för att läsa och skriva till filer och kommunikationsportar. Och kanske en massa annat.
  • Att göra dina egna Java applets och tillämpningar.
  • Att köra Windows Media Control Interface (MCI)
  • Vanliga frågor.
  • En sammanställning snabbkommandon på tangentbprdet.
  • Tillval för kommandon.
