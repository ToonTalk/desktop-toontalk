O que tentar se o Toon Talk está rodando
muito devagar

A velocidade com a qual o Toon Talk roda depende de muitas coisas. Estas incluem a velocidade de seu computador, a quantidade de memória que ele tem, que placas de som e vídeo você tem, que versão do Windows você está rodando, quão atuais são seus drivers de som e vídeo, entre outras coisas.

Se você instalou a partir do CD-Rom e escolheu a instalação compacta (ou às vezes mesmo a típica), algumas vezes o sistema se torna mais lento ao tentar encontrar arquivos no drive de CD-Rom. Reinstale e escolha a opção personalizada para ver se esse procedimento auxilia.

Em alguns computadores o Toon Talk roda em velocidades muito diferentes dependendo de você estar utilizando a versão de 16 ou de 32 bits. E para as versões de 32 bits os gráficos de alta resolução são tipicamente mais lentos. E algumas vezes a velocidade se altera drasticamente dependendo se o Toon Talk está rodando em tela cheia ou em uma janela. Ocasionalmente, o processamento para permitir que Márcio fale torna o computador claramente mais lento. Clique em "Set Options" depois de iniciar o Toon Talk e tente rodar versões diferentes dele.

A versão de 32 bits em um computador 486 é habitualmente vagarosa. Mesmo a versão de 16 bits roda devagar em alguns computadores 486 e na maioria das máquinas 386.

É possível que algum outro programa esteja rodando ao mesmo tempo e competindo com o Toon Talk por recursos. Você pode tentar rodar o Toon Talk isoladamente para ver se esse é o problema.
