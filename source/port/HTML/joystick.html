Utilizando joysticks com o Toon Talk

A versão de 32 bits do Toon Talk permite que você controle seu personagem utilizando um joystick ou controlador de jogo, se você assim preferir. Você pode precisar informar ao Windows 95 sobre seu joystick ou calibrá-lo, se funcionar corretamente. Clique aqui para aprender como fazê-lo.

Aqui está o que os diferentes botões do joystick fazem:

  1. Comporta-se como o botão esquerdo do mouse.
  2. Comporta-se como o botão direito do mouse (liga e desliga as coisas, move-as etc.)
  3. Por "default" comporta-se como quando se digita "- ".
  4. Por "default" comporta-se como quando se digita "F1".

Você pode alterar os "defaults" para os botões 3 e 4 e, se você tiver mais botões, adicionar equivalentes no teclado para eles, editando seu arquivo toon talk.ini

Você também pode ter seus jogos e outros programas utilizando até 16 joysticks conectados ao seu computador.
