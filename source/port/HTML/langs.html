O Toon Talk pode falar muitas línguas!

Bem, no momento, apenas três línguas.

  • Inglês (americano e britânico). A versão americana está incluída na versão beta de teste atual.
  • Sueco. Esta inclui traduções dos diálogos, tudo que Márcio diz, os quebra-cabeças e a documentação. Para instalar, apenas rode o arquivo swedish.exe no Windows. Para rodar o Toon Talk em sueco, vide abaixo. O CD-Rom do Toon Talk também inclui a narração em sueco para a maioria dos demos. O CD-Rom também permite que Márcio fale em sueco. A tradução foi feita por Lennart Mogren e Tobias Zetterlund. O Toon Talk está à venda em sueco.
  • Português (Brasil). Esta versão inclui, atualmente, traduções dos diálogos e de tudo que Marty diz. Para instalar, apenas rode o arquivo port.exe no Windows. A tradução foi feita por Leo Burd. NOTA: ESTA VERSÃO ESTÁ ATUALMENTE SOB REVISÃO. ENVIE UM E-MAIL PARA support@toontalk.com SE ESTIVER INTERESSADO.
  • Alemão. Em progresso. Logo todo o conteúdo deste Web Site terá sido traduzido. Clique aqui para explorar a tradução alemã.
  • Espanhol. Em progresso.
  • Sua língua aqui. Se você estiver interessado em ajudar a traduzir o Toon Talk para alguma outra língua, por favor nos informe.

Para utilizar uma língua instalada, apenas adicione a linha "- language Swedish" ou "- language Portuguese" à linha de comando para iniciar o Toon Talk.

Se você desejar simplificar a troca entre as línguas para o usuário, então você deve editar o arquivo toontalk.ini no diretório do Windows para adicionar a linha

Language2=Swedish

Depois da linha Language1=English

Depois de fazê-lo, o "Set Options", após iniciar o Toon Talk, oferecerá uma opção das línguas.
