Lidando com problemas utilizando o mouse com o Toon Talk

Quando o Toon Talk tenta descobrir como o mouse de seu computador se moveu, ele algumas vezes descobre que o mouse se moveu de uma forma impossível. Isto foi observado quando rodou-se o Toon Talk em um software como o Soft Windows ou o VirtualPC em um computador Macintosh.

Em um PC isso não deveria ocorrer, mas se ocorre apenas quando se move o mouse energicamente, então você pode querer reduzir a sensividade do mouse.

Você pode utilizar o Toon Talk sem mover o mouse. Em vez disso, pode utilizar em seu lugar as setas direcionais do teclado.
