Como lidar com uma parada súbita
do "Toon Talk"

Lembre-se de que o Toon Talk irá informar este problema se a máquina foi desligada da última vez que o programa estava rodando. Recomenda-se que você saia do Toon Talk pressionando a tecla "Pause" ou Alt-F4.

Se houver uma parada súbita do Toon Talk, existem diversas possibilidades. Primeiro, veja se ele pára novamente se você fizer a mesma coisa que antes.

  • O Toon Talk não está corretamente instalado. Isto não é provável se o Toon Talk estava funcionando corretamente antes. A reinstalação deve ser perfeitamente segura.
  • O Toon Talk roda em um sistema inadequado, com pouca memória ou espaço em disco. Certifique-se de que sua máquina não está totalmente cheia. Em alguns computadores os programas não rodam adequadamente se você tiver menos de 10 ou 20 megabytes de espaço livre em disco para utilizar o Windows.
  • Os arquivos que o Toon Talk mantém para um usuário particular podem estar corrompidos. Verifique se o ToonTalk funciona corretamente se você o informa com outro nome.
  • Você pode ter diferentes versões do Toon Talk disponíveis. Clique em "Set Options" (Opções de Ajuste) depois de começar o Toon Talk, e se ele perguntar que versão você deseja rodar, tente alterá-la para ver se funciona.
  • O Toon Talk pode rodar tanto no modo "Full Screen" (Tela Cheia) como em uma janela. Verifique se o problema persiste em ambos. Clique em "Set Options" depois de iniciar o Toon Talk para fazer a alteração.
  • A versão do Toon Talk de 32 bits utiliza as extensões para games DirectX da Microsoft. Se você estiver rodando a versão de 32 bits, certifique-se de que possui a última versão do DirectX e seus drivers. Você pode encontrar mais informações em www.microsoft.com/directx.
  • A versão de 16 bits pode apresentar problemas rodando em algumas placas de som mais antigas. Tente desligar os efeitos sonoros clicando em "Set Options" depois de iniciar o Toon Talk.
  • Em alguns computadores e opções de ajuste do Toon Talk a tela pode ficar escura ou parecer nublada ou estática. Ou o Toon Talk pode travar aleatoriamente. Se o Windows estiver rodando com mais de 256 cores, tente mudar para 256 cores. Você pode fazê-lo através do "Painel de Controle".
