Exemplos do Toon Talk

Se você clicar em "Veja Demos" depois de iniciar o Toon Talk, você verá uma seleção de oito demos.

  1. Tour Introdutório (menos de 15 minutos). Esta é uma ampla introdução ao Toon Talk, seus objetos e ferramentas e exemplos de programações simples.
  2. Explodindo Casas (menos de 5 minutos). Este é o mais simples dos demos e provavelmente é o mais popular.
  3. Troque se for maior (menos de 3 minutos). Este demo ilustra duas técnicas de programação extremamente básicas: condicionais de comparação e troca de valores em dois ambientes.
  4. Construir Pingue-Pongue (cada parte com menos de 20 minutos). Este demo mostra dois garotos tentando construir um jogo de pingue-pongue. Eles não sabem o que estão fazendo e o demo mostra o processo de exploração, tentativa, descoberta de erros, conserto dos erros, obtenção de ajuda e outros semelhantes.
  5. Números de Fibonacci (menos de 4 minutos). Este demo ilustra o tipo de exploração matemática que é possível no interior do Toon Talk.
  6. Fatoriais (menos de 6 minutos). Este demo exibe uma técnica de programação que é crítica para a construção de grandes programas no Toon Talk fora das partes que funcionam em conjunto. A técnica é uma generalização de procedimentos de definição e chamada.
  7. Conta Bancária (menos de 15 minutos). Este demo exibe como conceitos avançados de programação de computadores e de ciência da computação podem ser tornados acessíveis no interior do Toon Talk. Ele também ilustra outra importante técnica de programação necessária para programação em larga escala – objetos concorrentes com o estado.
  8. Anexação de listas (menos de 8 minutos). Este demo ilustra outra técnica de programação avançada (manipulação de listas) e ilustra a conexão entre o Toon Talk e a lógica da programação.
