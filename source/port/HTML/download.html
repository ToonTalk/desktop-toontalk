Lançamento da versão Beta 14B do Toon Talk disponível em 16 de outubro de 1998

O Beta 14b está disponível apenas como um grande (8mb) arquivo executável de auto-extração chamado bta14b16.exe. Ele é protegido por senha – mande um e-mail para solicitar a senha. A versão de teste do Toon Talk não exige senha.

Para instalar.

Copie o arquivo para um diretório temporário (por exemplo \windows\temp). Então, rode o bta14b16. Será solicitada uma senha que você pode obter agora através do envio de um e-mail.

Há também um arquivo readme.txt que você pode querer ler primeiro.

A versão Beta 14B expirará 60 dias após sua instalação.

Localização de problemas na instalação.

Se seu navegador estiver rodando em uma estação de trabalho Sun (ou algumas outras máquinas Unix), você pode ter dificuldades ao transferir os arquivos acima. Você pode solucionar o problema utilizando um ftp anônimo em ftp.toontalk.com/ftp/TTBeta14.

Se você encontra erros durante a instalação, é possível que o arquivo não tenha sido copiado perfeitamente para seu disco rígido. O arquivo bta14b16.exe deve ter exatamente 8.217.652 bytes de extensão. Se não for o caso, tente baixar o arquivo novamente.

Outras versões do Toon Talk.

Existe uma versão de 32 bits do Toon Talk que pode rodar em Windows 95 e Windows NT 4.0. Você precisa instalar a versão de 16 bits primeiro. Há atualmente um problema rodando a versão de 16 bits do Toon Talk em Windows NT 4.0. Se a versão de 16 bits se recusa a rodar você precisará instalar também a versão de 32 bits. O bta14b32.exe depois do download deve ter 1.205.790 bytes de extensão.

A instalação não inclui nenhuma narração. Se você quiser ouvir narrações dos demos, clique aqui para mais informações.

O Toon Talk está também disponível em Sueco e Português.

A instalação precisa desses dois arquivos apenas para o Windows NT 3.51
