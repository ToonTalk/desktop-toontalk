Lidando com problemas de DirectX
no Toon Talk

A versão de 32 bits do Toon Talk se baseia nas extensões para jogos DirectX da Microsoft para Windows. Por favor, assegure-se de ter a última versão do DirectX e seus drivers. O CD-Rom do Toon Talk instala a versão 5 do DirectX. Mais informações sobre o DirectX podem ser obtidas em www.microsoft.com/directx.

Se o problema persistir quando estiver rodando a versão de 32 bits do Toon Talk, e estiver ausente quando estiver rodando a versão de 16 bits ( a qual utiliza as extensões para jogos mais antigas da Microsoft, chamadas WinG) e você tiver instalada a última versão do DirectX, por favor relate o fato para support@toontalk.com. Você pode trocar entre as versões de 16 e de 32 bits do Toon Talk através da seleção de "Set Options" depois de iniciar o Toon Talk.

Se a versão de 32 bits do Toon Talk funcionou previamente, então ao reiniciar seu computador você algumas vezes corrigirá este tipo de defeito do DirectX.

Em algumas situações, a versão de 32 bits do Toon Talk não funcionará se o sistema estiver ajustado para mais de 256 cores. Você pode mudar o número de cores através do Painel de Controle em Meu Computador.
