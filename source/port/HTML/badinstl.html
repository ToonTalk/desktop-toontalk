Lidando com problemas de instalação
do Toon Talk

Você percebe que parece haver algo de errado com a instalação do Toon Talk em sua máquina. Se você tentou copiar arquivos a partir de outra instalação, é esperado que isso ocorra. Você precisará reinstalar o Toon Talk a partir do CD-Rom, se você possuí-lo, ou o arquivo que foi baixado da Internet.

A instalação pode falhar se você tiver muito pouco espaço em disco. Normalmente, você receberá uma mensagem de aviso dizendo que possui muito pouco espaço disponível no disco, mas não sempre se, por exemplo, você estiver rodando com a compressão do disco ativada.

Se você instalou o Toon Talk a partir de um arquivo que baixou da Internet, então cheque se o tamanho é correto.

Clique aqui para relatar um problema.

Se você possui uma conexão com a Internet, você pode clicar aqui para ver as últimas informações acerca desse assunto.
