Asper, o aspirador de pó, suga coisas e
as elimina novamente.

Se você apertar a barra de espaço enquanto segura Asper, ele se voltará e sugará coisas até que você pressione novamente a barra de espaço.

Se você apertar o botão em seu nariz, ele mudará de "A" para "D" e para "I", e novamente para "A". A é para aspirar, D para devolver e I para limpar.

Para usuários avançados há também comandos especiais de teclado que você pode utilizar enquanto o segura.
