Animated Programs apresenta o Toon Talk

software revolucionário que ensina programação de computadores para crianças

Para lançamento imediato:

SAN MATEO, Califórnia - 11 de janeiro, 1999 - A Animated Programs anunciou hoje o lançamento do Toon Talk, um programa de computador educacional que permite que as crianças aprendam e utilizem programação de computadores.

O Toon Talk é um novo meio de construir, corrigir e rodar programas. No Toon Talk, os programas não são construídos digitando algum texto ou reunindo imagens. Eles são construídos executando ações em um mundo animado interativo. Objetos familiares, tais como pássaros, robôs, caminhões e bombas de bicicleta são as únicas ferramentas de programação necessárias. O Toon Talk torna a programação fácil de compreender para as crianças, fazendo-a tangível e parecida com um jogo. Enquanto aprendem como programar, as crianças enfrentarão quebra-cabeças desafiadores, expressarão sua criatividade, aprenderão novos meios de resolver problemas e, acima de tudo, se divertirão!

Aprendendo a programar:

Muitos educadores questionam o valor de ensinar a programação, porque ela é difícil de aprender e difícil de fazer. Agora, com o Toon Talk, tanto as crianças quanto os adultos podem aprender a programar simplesmente aprendendo o comportamento de objetos familiares e utilizando-os para criar seus próprios programas. O Toon Talk inclui um novo modo de aprender programação – jogando um game. No jogo, os jogadores auxiliam um marciano ferido chamado Márcio a reparar sua nave espacial. De forma a ajudar Márcio, eles precisam resolver uma seqüência de quebra-cabeças que introduzem gradualmente conceitos e técnicas de programação. Os usuários são também apresentados a objetos familiares tais como caixas, casas, robôs, estradas, aspiradores de pó etc. e como eles funcionam no interior do Toon Talk. Os primeiros três quebra-cabeças introduzem números, adição e caixas (estrutura de dados). O quarto quebra-cabeça começa com a introdução de objetos familiares tais como uma varinha mágica e casas. No oitavo quebra-cabeça, o jogador utilizará robôs para fazer o primeiro programa real. Em qualquer momento, durante os quebra-cabeças, se uma criança está confusa ou precisa de ajuda, Márcio está pronto e determinado a oferecer dicas ou instruções completas, se necessário. O Toon Talk também inclui demonstrações de programação narradas que mostram como construir programas.

O Toon Talk foi desenvolvido para crianças e fornece três meios fundamentalmente diferentes para que as crianças aprendam:

    Free play (Jogo Livre) - Um ambiente ilimitado, não restrito, rico, para explorar e criar coisas.
    Um jogo de quebra-cabeça - Uma seqüência de quebra-cabeças que gradualmente introduzem técnicas para a construção de programas.
    Demostrações - Demonstrações narradas de várias técnicas de construção e conserto de programas.

O Poder da Programação

O Toon Talk é uma linguagem de programação de uso geral. Ele tem uma propriedade chamada "universalidade", assim sendo, em princípio, qualquer programa pode ser criado no Toon Talk. A maioria dos programas que as crianças apreciam são jogos, tais como Pong, Pacman, Break Out, Missile command etc. Elas também podem criar animações, explorações matemáticas, programas que geram sentenças e simulações. Uma vez que as crianças tenham criado seus programas, elas podem compartilhá-los com outras pela Web. O Toon Talk gera um código de fonte Java que é equivalente ao que as crianças criaram no Toon Talk. Elas podem então rodar seus programas como um aplicativo no navegador da Web.

Projetado para crianças

Ao se projetar uma linguagem de programação específica para as crianças, foram tomados cuidados para não diminuir o poder de programação. A linguagem subjacente do Toon Talk é baseada em anos de pesquisa feitos por um grupo de cientistas da computação da Xerox PARC. As crianças dominam o Toon Talk compreendendo os equivalentes do programa para canais de comunicação e sincronização de processos concorrentes, tais como: os pássaros levam mensagens para seus ninhos, e os robôs esperarão até que algo chegue ao ninho antes de fazer o que foram treinados para realizar. As crianças nem mesmo imaginam que compreendendo os pássaros e robôs do Toon Talk elas estão aprendendo como programar.

Sobre o criador do Toon Talk

O Toon Talk foi criado e construído por Ken Kahn, um Ph.D em ciência da computação pelo MIT com mais de 20 anos de experiência em projetos de linguagem de programação na Xerox PARC e em várias universidades. Kahn recebeu uma patente para interfaces animadas de usuário para a criação de programas de computador, controle e execução da tecnologia subjacente ao Toon Talk.

Sobre a Animated Programs

A Animated Programs, sediada em San Mateo, Califórnia, foi fundada em 1992 com a missão de tornar a programação de computadores brincadeira de criança. O primeiro programa oferecido pela companhia é o Toon Talk, um software que torna a programação divertida e fácil de aprender.
