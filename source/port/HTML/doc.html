Um guia para os jogadores do Toon Talk

Quando você clica no ícone do Toon Talk, tem as seguintes opções:

  1. Jogar os Desafios. Esta é uma forma divertida de aprender como as coisas funcionam no Toon Talk e como construir coisas interessantes. Depois de resolver alguns dos desafios, você pode querer saber o que você aprendeu.

  2. Veja os exemplos. Você pode apenas se sentar e observar um tour introdutório narrado, ou observar como dois garotos tentam construir um jogo de pingue-pongue, ou assistir a vários outros demos. Clique aqui para mais informações.

  3. Jogo Livre. Você começará voando em seu helicóptero sobre a ilha de Toon Talk. Márcio irá auxiliá-lo assim que você entrar na casa.

  4. Opções de Ajuste. Permite que você personalize o Toon Talk para o seu computador.

  5. Ajuda. Inicia seu navegador lendo esse arquivo.

Busca por:


Você pode utilizar os operadores booleanos e, ou ou não (estes são os casos mais primários). Sem eles o sistema de busca suporá que você está utilizando e entre as palavras.
