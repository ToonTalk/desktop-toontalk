O que fazer se o Toon Talk não está funcionando corretamente

Se você acha que o Toon Talk não está funcionando corretamente, por favor clique em um dos problemas listados abaixo. Se o problema persistir ou não estiver citado aqui, por favor, reporte-o.

  • O Toon Talk está parando ou travando.
  • O Toon Talk está rodando muito devagar.
  • O Toon Talk está tendo problemas ao rodar demos.
  • O Toon Talk não instala corretamente.
  • O Toon Talk parece ter problemas com as extensões para jogos da Microsoft DirectX.
  • A versão de 16 bits está tendo problemas rodando em tela cheia.
  • O Toon Talk está tendo problemas ao utilizar meu joystick ou controlador
    de jogos    .

Clique aqui para relatar um problema.
