Dicas básicas para os quebra-cabeças
do Toon Talk

Quando estiver jogando o jogo de quebra-cabeças do Toon Talk, a primeira coisa que você deve fazer é mover o mouse para entrar no foguete, atravessando a porta. Márcio o recepcionará e explicará o primeiro problema. Você então deverá caminhar de volta através da porta do foguete e voltar para a porta da casa. Uma vez dentro dela, aperte o botão do mouse para sentar-se. Então construa o que quer que Márcio precise, utilizando o botão do mouse para apanhar e soltar coisas. Se você fizer bagunça, apanhe a bomba e aperte a barra de espaço. Se você precisar de ajuda, volte para Márcio. Cada vez que você entrar no quarto ou bater nele, você terá novas dicas de Márcio. Quando achar que construiu tudo de que Márcio necessita, apanhe as coisas e levante-se, pressionando a tecla "Esc" do teclado. Então, caminhe de volta para o foguete.
