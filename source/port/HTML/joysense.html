Sensores de joystick no Toon Talk

Se você tem um joystick conectado ao seu computador e está rodando a versão de 32 bits do Toon Talk no Windows 95, então você encontrará uma agenda de sensores para cada joystick que você conectou. Aqui está o que são os sensores:

  • Página 2. Joystick X Velocidade. Este sensor exibe quanto, para a esquerda ou direita, o joystick se moveu. Varia de -1.000 para 1.000.
  • Página 4. Joystick Y Velocidade. Este sensor exibe quanto, para a frente ou para trás, o joystick se moveu. Varia de -1.000 para 1.000.
  • Página 6. Joystick Z Velocidade. Este sensor exibe quanto, para cima ou para baixo, o joystick se moveu. Ele é geralmente o controle do acelerador. Se o joystick não tiver esse dispositivo, o valor é zero. Varia de -1.000 para 1.000.
  • Página 8. Joystick X Eixo. Este sensor exibe quanto, para a direita ou esquerda, o joystick foi rodado. Se o joystick não tiver esse dispositivo, o valor é zero. Varia de -1.000 para 1.000.
  • Página 10. Joystick Y Eixo. Este sensor exibe quanto o joystick foi rodado, para a frente ou para trás. Se o joystick não tiver esse dispositivo, o valor é zero. Varia de -1.000 para 1.000.
  • Página 12. Joystick Z Eixo. Este sensor exibe quanto o joystick foi rodado, para cima ou para baixo. Ele é geralmente chamado de direção. Se o joystick não tem esse dispositivo, o valor é zero. Varia de -1.000 para 1.000
  • Página 14. Cliques dos botões do Joystick. O Toon Talk pode reconhecer até 32 botões de joystick. Quando um botão é clicado, o valor se torna, rapidamente, "sim". De outra forma, é "não".
  • Página 16. Botões do Joystick abaixados. Isto exibe "sim" se o botão correspondente do joystick estiver abaixado, de outra forma, exibe "não".

Se você tem um joystick force feedback, então haverá, também, uma agenda especial para fazer e controlar os efeitos de força.
