Como instalar extensões do Toon Talk

Apenas para usuários avançados

As extensões do Toon Talk não são instaladas por "default". Uma razão para isso é que elas permitem que os robôs troquem coisas fora do mundo seguro do Toon Talk. As extensões de arquivos, por exemplo, permitem que os robôs do Toon Talk leiam e alterem (ou destruam) qualquer arquivo em seu computador. Este poder pode ser usado para o bem ou para o mal. Se você instalar essa extensão, então certifique-se de que você confia que lhe deu os programas do Toon Talk para rodar.

Atualmente, as extensões só funcionam na versão de 32 bits do Toon Talk. Se você está interessado em rodar extensões na versão de 16 bits, envie um e-mail para support@toontalk.com.

Uma extensão está instalada se o arquivo DLL começando com "TT" seguido pelo nome da extensão está em seu diretório Toon Talk. Por exemplo, o arquivo de extensão do arquivo é "TTFile.dll".

Se você estiver instalando a partir de um CD-Rom, então escolha a opção "Custom Installation" (Instalação Personalizada). Selecione "Arquivos de Programas" e clique no botão "Alterar". Você deve então desativar todos os componentes, exceto aqueles que são nomeados com as extensões que você deseja. De forma alternativa, copie os arquivos apropriados do subdiretório \toontalk\extend.

Se você estiver instalando a partir da Internet, então baixe e copie o TTFile.dll para seu diretório Toon Talk para instalar a extensão de arquivo.
