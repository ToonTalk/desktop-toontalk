O Limpopó, o Aspirador, aspira coisas e cospe-as.

Se carregares na barra de espaços, enquanto seguras o Limpopó, ele liga-se e aspira coisas até voltares a carregar na barra de espaços outra vez.

Se carregares no botão que está no seu nariz, ele muda de 'A' para 'L' e para 'C'. 'A' é para aspirar, 'L' para limpar e 'C' para cuspir.

Para os utilizadores mais avançados, existem também os comandos especiais do teclado que podes usar, enquanto seguras no Limpopó.
