Manual para os Jogadores do ToonTalk

Ao clicares no ícone do ToonTalk , aparecem as seguintes opções:

  1. Puzzles.  Esta é uma maneira engraçada de veres como as coisas funcionam e como se podem construir coisas. Depois de resolveres alguns puzzles, podes querer saber o que aprendeste.
  2. Exemplos.  Observa as demonstrações e aprende como construir os teus próprios jogos e programas. Clica aqui para mais informações.
  3. Brincar.  Começarás por sobrevoar a ilha Zizzle com o teu helicóptero. O ET vai ajudar-te, assim que entrares numa casa.
  4. Opções. Altera o modo de funcionamento do ToonTalk.
  5. Ajuda. Inicia o navegador da Internet que lê este ficheiro.

Se quiseres saber mais coisas sobre qualquer um dos tópicos que se seguem, basta clicares nas palavras sublinhadas.

  • Trabalhar com o ET, o extra-terrestre.
  • Usar o Limpopó, a Artolas e a Varinha Mágica.
  • Blocos de números e texto.
  • Caixas.
  • Pássaros e ninhos.
  • Balanças para comparar coisas.
  • Treinar robots.
  • Camiões para construir casas novas.
  • Bombas para destruir casas.
  • O teu caderno.
  • Usar imagens e animações.
  • Usar efeitos sonoros.
  • Voar e aterrar o helicóptero.
  • Sentar e levantar.
  • Fazer coisas no chão.
  • Configurar o ToonTalk.
  • Usar sensores.
  • Usar as extensões do ToonTalk para ler e escrever em ficheiros e portas de comunicação.
  • Criar as tuas próprias Java applets e aplicações.
  • Executar o Media Control Interface (MCI) do Windows
  • Perguntas mais frequentes.
  • Síntese de todos os comandos do teclado.
  • Opções da linha de comando.
