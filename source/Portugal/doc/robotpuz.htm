Dicas para usar Robots nos Puzzles do ToonTalk

No ToonTalk, um robot pode ser treinado dando-lhe uma caixa. A seguir entras no seu balão de pensamento. O que quer que faças, o robot irá lembrar-se. Alguns robots apenas conseguem lembrar-se de alguns passos. Quando a memória de um robot já está cheia, sais do seu balão de pensamento. Podes sair quando quiseres, carregando na tecla 'Esc'.

Assim que um robot estiver treinado, podes dar-lhe uma caixa. Se a caixa coincidir com o que está no seu balão de pensamento, o robot repetirá aquilo que foi treinado para fazer. Se a caixa não coincidir com o que ele está a pensar, aparece o Limpopó. Podes usar o Limpopó para aspirar coisas do balão de pensamento do robot, para ele não ficar tão confuso. O que deves fazer é aspirar apenas o suficiente para o robot ficar contente com o que tu lhe queres dar.

Alguns robots têm uma varinha mágica, que apenas eles podem usar. Terás de ensinar o robot a usar a varinha mágica.
