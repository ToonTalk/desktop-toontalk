Exemplos Java feitos em ToonTalk

ToonTalk tem um tradutor para Java. Qualquer coisa feita em ToonTalk pode ser automaticamente convertida em Java e ser executada no teu navegador da Internet. Aqui estão alguns exemplos:

  • Sequência de Fibonacci. Este exemplo cria uma sequência de números onde cada número é a soma dos dois anteriores.
  • Factorial. Este exemplo calcula o factorial de um número.
  • Jogo Ping Pong. Este exemplo executa o jogo simples do Ping Pong.
  • Chase Movies. Este exemplo mostra o resultado da combinação de comportamentos em ToonTalk. Há objectos que fazem ricochete em paredes, uns contra os outros, perseguem outros, etc...
