ToonTalk e Java Applets

ToonTalk pode criar Java applets a partir de um programa do ToonTalk.

Para criar uma Java applet devemos pegar em qualquer coisa, carregar na tecla pause, e depois em "Voltar mais tarde". Se tudo correr bem o navegador (Internet Explorer, por ex.) deverá ser iniciado e deverás ver o teu programa a correr. Para experimentares vai ao livro de exemplos na página 6 e apanha o robot que repetidamente soma 1, na página 2. Carrega em pause e carrega em "Voltar mais tarde". Quando a applet aparecer no teu navegador da Internet, clica em "Começar" e vê o número a crescer. Só os robots, camiões e imagens podem criar Java applets.  Quando carregas um camião podes te que carregar em F8, para impedir que o camião se vá embora. Podes por um bloco de texto para dar um nome à applet. Se quiseres que os robots utilizem cadernos especiais, tens de os colocar no camião. Para dar um nome à applet criada a partir de imagens (geralmente com robots nas costas) coloca um bloco de texto nas costas da imagem.

Uma applet pode pedir valores ao utilizador. Se utilizares um ninho numa caixa (o pássaro desse ninho não vai pertencer à Java applet) e puseres uma etiqueta no buraco onde está o ninho, então a Java applet vai precisar de um valor e pergunta ao utilizador o valor, escrevendo a etiqueta. Igualmente, se um pássaro estiver num buraco com uma etiqueta e lhe for dado algo, então a etiqueta é escrita juntamente com a descrição do que foi dado ao pássaro. A ideia é que os pássaros e os ninhos servem para comunicar com o utilizador numa Java applet.

Isto só possível em navegadores que suportem Java applets, como o Microsoft Internet Explorer 4.0 ou superior ou o Netscape Navigator 4.0 ou superior. 

Microsoft Internet Explorer (versão 4.0 SP1) (parte da primeira edição do Windows 98) por vezes dá "Invalid Page Fault" quando tenta executar applets criadas pelo ToonTalk. Por vezes bloqueia ou tem erro do Java. Podes seleccionar "Opções da Internet" no menu "Ver", clicar no Avançado, procurares "Compilador JIT para Java activado". Certifica-te que não fica seleccionado. Isto deve resolver o problema, mas as applets podem ficar mais lentas. Por favor, se continuares a ter problemas contacta-nos.
