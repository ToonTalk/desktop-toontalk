Usar Joysticks com o ToonTalk

A versão de 32 bits do ToonTalk deixa que tu controles a tua personagem usando um joystick ou um controlador de jogos, se preferires. Podes precisar de explicar ao Windows 95 como é o teu joystick ou calibrá-lo para que funcione bem. Clica aqui para aprenderes a fazê-lo.

Aqui está o que fazem os diferentes botões do joystick:

  1. Funciona tal como o botão esquerdo do rato.
  2. Funciona tal como o botão direito do rato (liga ou desliga as coisas, vira-as, etc.). 
  3. Normalmente comporta-se como se carregasses na tecla '-'.
  4. Normalmente comporta-se como se carregasses na tecla 'F1'.

Podes redefinir as funções dos botões 3 e 4. Se tiveres mais botões, podes acrescentar equivalentes do teclado a esses botões, editando o teu ficheiro ToonTalk.ini .

Podes fazer com que os teus jogos e programas usem até  16 joysticks ligados ao teu computador.

Se tiveres um joystick force feedback, poderás sentir as paredes, o helicóptero a voar, ferramentas, etc. E podes programar o teu joystick a partir do ToonTalk.
