As Balanças comparam coisas

Dentro da Engenhocas, vais encontrar um grande número de balanças. Podes usá-las para descobrir qual é o número maior ou que palavra ou frase vem primeiro por ordem alfabética. Larga a balança num buraco de uma caixa com buracos dos dois lados. Se puseres números nesses buracos, a balança (que é uma balança) fica mais pesada do lado do número maior. Se puseres blocos de texto, a balança fica mais pesada do lado do texto que aparece depois no alfabeto.

Começando com a versão 1.1 (ou beta 15), as balanças podem também ser usadas para comparar imagens. As imagens do utilizador são comparadas por ordem alfabética dos nomes dos ficheiros. As imagens do utilizador são "mais pesadas" do que as imagens já existentes no caderno do ToonTalk. Estas imagens são comparadas através de um índice de comparação interno.

Dica Avançada. Podes treinar um robot para fazer qualquer coisa, se um número for maior do que outro, e outro robot para fazer outra coisa, se o número for menor (e ainda outro robot para fazer qualquer coisa, se os números forem iguais). O exemplo "Mudar se maior" mostra como um robot pode ser treinado para trocar dois números, se o primeiro for menor que o segundo. Tem em atenção que a balança vacila para a frente ou para trás se estiver perto de um buraco vazio. Por isso é importante usar o Limpopó para apagar, não para aspirar, os blocos de texto que estão perto da balança, quando queres que o teu robot trabalhe com mais tipos de caixas.
