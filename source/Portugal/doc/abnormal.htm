Como resolver problemas do ToonTalk

Atenção que o ToonTalk mostra este problema, se desligares o computador enquanto estavas no ToonTalk. Recomendamos que saias carregando na tecla "Pause" ou Alt-F4.

Se o ToonTalk pára de repente, existem várias explicações. Primeiro vê se pára outra vez, se fizeres a mesma coisa.

  • O ToonTalk não foi instalado como deve ser. Isto é pouco provável se o ToonTalk já esteve a funcionar. Reinstalando-o deve resolver o problema.
  • O ToonTalk ficou sem recursos do sistema, como a memória ou o espaço em disco. Certifica-te de que o computador não está completamente cheio. Nalguns computadores, os programas não correm correctamente se tiveres menos de 10 ou 20 megabytes de espaço livre para o Windows utilizar.
  • Os ficheiros que o ToonTalk mantém para um utilizador podem estar estragados. Experimenta utilizar outro nome para ver se o ToonTalk já funciona correctamente.
  • Tens diferentes versões do ToonTalk disponíveis. Clica em "Opções" depois de iniciares o ToonTalk e, se te for perguntado que versão queres utilizar, tenta mudar para ver se resolve o problema.
  • O ToonTalk pode ser executado em ecrã inteiro ou numa janela. Experimenta mudar o modo. Clica em "Opções" para mudares isto.
  • O ToonTalk utiliza as extensões DirectX. Certifica-te que tens a última versão do DirectX e dos drivers. Podes encontrar mais informação em www.microsoft.com/directx.
  • Se vires uma janela grande branca e se te parecer que nada está a funcionar, carrega na tecla 'Enter' por que o mais certo é ser uma caixa de diálogo que está por trás a fazer uma pergunta.
  • Nalguns computadores, o ecrã pode ficar preto, como neve, estático ou o ToonTalk bloqueia aleatoriamente. Se o Windows estiver com mais de 256 cores, tenta mudar para 256. Podes fazer isto no Painel de Controlo.
