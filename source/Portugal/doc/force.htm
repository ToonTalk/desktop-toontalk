Como Programar o teu Joystick Force Feedback 

 

Apenas para a versão de 32 bits.

Só para utilizadores mais avançados:

Se tiveres um joystick force feedback  (e ele for compatível com o DirectX 5 ou posterior; por exemplo, Microsoft's Sidewinder Force Feedback Pro or CH Products Force FX), deverás procurar um caderno de controlos remotos especiais para o teu joystick (este caderno está perto do final do caderno dos sensores). Neste caderno, irás encontrar efeitos de força nas páginas ímpares e alguns controladores nas páginas pares. Um efeito de força é ligado ou desligado carregando na barra de espaços, enquanto seguras nele. Como sempre, os robots podem ser treinados para fazer isto.

Os três controladores para um efeito de força são:

  1. Duração -  Este determina quanto tempo o efeito irá durar. É medido em milissegundos (milésimas de segundo).
  2. Força - Este determina quanta força tem o efeito. 100 é um efeito 100% cheio de força.
  3. Direcção - Este determina a direcção que a força vai tomar. É medida em graus (por isso varia entre 0 e 359).

Para ler os botões ou o movimento do joystick, usa os sensores que estão no caderno normal do joystick.

Os efeitos de força foram criados usando IFORCE Studio, que está disponível gratuitamente a partir da Immersion Corporation. Podes alterar qualquer uma das forças existentes ou criar novas. Elas ficarão no teu caderno na próxima vez que iniciares o ToonTalk.

Tem atenção que o ToonTalk, normalmente, desliga uma ferramenta chamada Autocenter. Ele usa os motores, para simular as molas num joystick normal. Desligando-o, tens maior controlo sobre ele. O TurnOffJoystickAutoCenter flag no teu ficheiro toontalk.ini pode ser configurado para ligar o Autocenter.
