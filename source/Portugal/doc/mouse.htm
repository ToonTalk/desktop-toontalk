Usar o Rato no ToonTalk

Para simplificar as coisas, normalmente todos os botões do rato têm as mesmas funções no ToonTalk. Podes configurar aceleradores para os botões do meio e direito do rato. O botão direito do rato pode ser configurado para que, quando se clica nele enquanto se segura numa ferramenta, se possa usá-la. Por outras palavras, funciona como a "barra de espaços". O botão direito do rato também pode ser utilizado para virar texto, números e imagens. O botão do meio pode ser configurado para qualquer tecla. A maneira mais fácil para alterar isto é seleccionar a opção que queres depois de clicares em "Opções" (ao iniciares o ToonTalk).

A configuração normal para o botão do meio é '-', para veres as páginas anteriores dos teus cadernos e para andar ao contrário nos ciclos de alguns controlos remotos. Isto pode ser configurado no teu ficheiro toontalk.ini.

Se preferires não utilizar o rato (ou usá-lo apenas para voar e para andar), podes escrever "-b 0" na linha de comando. Isto permite-te utilizar vários aceleradores do teclado.
