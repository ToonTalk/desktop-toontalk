Utilizar Sensores e Controlos Remotos

Se virares uma imagem, verás um caderno a voar. No caderno encontrarás sensores e controlos remotos para a imagem. Os sensores mostram-te coisas sobre a imagem. Os controlos remotos deixam-te alterar coisas. Algumas vezes, eles aparecem combinados. Os controladores são o que nós chamamos sensores, controlos remotos e combinações. Os sensores especiais estão num caderno de sensores especial.

Muitos dos controladores mostram um número que podes mudar como qualquer outro número. E quando fazes alguma coisa com a imagem, ele também muda. Quando mudas a imagem, o número também muda automaticamente. Alguns controladores mostram texto e imagem. Podes mudar muitos deles carregando na barra de espaços, enquanto seguras neles ou apontas para eles.

Dica: Os controladores mudam ou alteram os sentidos da imagem quando estão nas suas costas. Se eles nunca estiveram nas costas de uma imagem, então eles devem ter vindo de um caderno que vive nas costas da imagem. Assim, eles controlam essa imagem.

Dica: Para experimentares o que um controlador faz, tira-o e põe-no no chão. Vira a imagem ao contrário (para poderes ver as suas costas) e põe-na também no chão. Muda o controlador e vê o que acontece.

Aqui está uma lista de controladores que encontrarás no caderno de uma imagem:

Página 2 - Do lado esquerdo
Distância da imagem a partir do lado esquerdo. Se a imagem não for parte de outra imagem, então 0 é a margem esquerda do ecrã e 1000 é a margem direita. Se a imagem foi acrescentada a outra imagem, então 0 é a margem esquerda da imagem à qual a outra imagem foi acrescentada e 1000 é a margem direita.
Página 4 - De baixo
Distância da imagem a partir de baixo. Se a imagem não for parte de outra imagem, então 0 é a margem inferior do ecrã e 1000 é a margem superior. Se a imagem foi acrescentada a outra imagem, então 0 é a margem inferior da imagem à qual a outra imagem foi acrescentada e 1000 é a margem superior.
Página 6 - Velocidade para a Direita
Velocidade à qual a imagem se move para a direita. 1000 significa que ela atravessa o ecrã num segundo. Se o número for negativo, a imagem desloca-se para a esquerda.
Página 8 - Velocidade para Cima
Velocidade à qual a imagem se move para cima. 1000 significa que ela atravessa o ecrã num segundo. Se o número for negativo, a imagem desloca-se para baixo.
Página 10 - Largura
Largura da imagem. 1000 significa que a imagem fica tão larga quanto o tamanho do ecrã. Ou se a imagem for uma parte de outra, ela ficará tão larga quanto a outra imagem.
Página 12 - Altura
Altura da imagem. 1000 significa que a imagem fica tão alta quanto o tamanho do ecrã. Ou se a imagem for uma parte de outra, ela ficará tão alta quanto a outra imagem.
Página 14 - Que imagem
Algumas imagens ou animações têm diferentes aparências. Algumas têm 4 direcções diferentes para se moverem. Este controlador mostra o número da imagem que está a ser mostrado. Para os rectângulos, círculos e linhas, ele controla a cor das formas geométricas. Para blocos de texto e de números, ele controla a cor dos caracteres.
Página 16 - Colidir?
Este controlador mostra um boneco simples de uma colisão ou de uma falha. Podes transformar uma colisão numa falha, apontando para o detector e carregando na barra de espaços.
Página 18 - Bater em quem?
Este sensor mostra com quem a imagem vai colidir. É preto, se não existir nenhuma. Se a imagem colidir com mais do que uma, é mostrado. Podes usar o Limpopó para aspirar a imagem que está por cima das outras. Também podes virar as imagens que aspiraste. Terás uma cópia da imagem que está a colidir. Se existir um um pássaro nas costas, isto pode ser muito útil.
Página 20 - Bater esquerda ou direita?
Funciona como o controlador da página 16, só que mostra a colisão, se tiver sido mais o movimento esquerdo ou direito da imagem que a causou.
Página 22 - Bater cima ou baixo?
Funciona como o controlador da página 16, só que mostra a colisão, se tiver sido mais o movimento para cima ou para baixo da imagem que a causou.
Página 24 - Visível?
Este controla quando é que a imagem está visível ou invisível. E também quando é que o fundo está visível ou não.
Página 26 - Animação Terminada?
Este sensor responde à pergunta: "A minha animação já parou?". Algumas animações, como andar (por exemplo), nunca param. outras, como as explosões (por exemplo), para e não se repetem.
Página 28 - Seguro na mão?
Este sensor responde à pergunta: "A imagem está segura na mão?".
Página 30 - Seleccionada?
Este sensor responde à pergunta: "A imagem está seleccionada e pronta para ser segurada ou usada?".
Página 32 - Largada agora?
Este sensor responde à pergunta: "A imagem acabou de ser largada?".
Página 34 - Aspecto
Este sensor mostra qual é o aspecto corrente da imagem. Tu (ou um robot que tenhas treinado) podes largar uma imagem diferente (ou um bloco de texto ou de números) aqui e a imagem será coberta pela nova imagem. É óptimo para fazer explodir objectos ou números que mudam quando muda a pontuação, ou para muitos outros tipos de utilizações engraçadas.
Página 36 - Partes
Este sensor contém uma caixa que contém todas as imagens que estão directamente em cima desta imagem. Uma imagem por buraco. Podes aspirar a caixa e manipulá-la. Esta ferramenta só está disponível na versão 1.1 ou Beta 15 ou posterior.
Página 38 - Contentores
Este sensor contém uma caixa. Se a imagem for uma parte de outra imagem, o primeiro buraco da caixa irá conter essa imagem. O segundo buraco contém a imagem à qual foi acrescentada a outra imagem, e por aí fora. Podes aspirar a caixa e manipulá-la. Esta ferramenta só está disponível na versão 1.1 ou Beta 15 ou posterior.
