Instalação do ToonTalk num Servidor de rede

A instalação num servidor de rede é mais complicada do que devia ser, mas funciona:

  1. Copiar todo o  CD-ROM para uma directoria no servidor. (No final podes apagar grande parte).
  2. Substituir o ficheiro setup.ins na directoria  Install pela  nova versão, cujo download pode ser feito a a partir da página do ToonTalk. serão cerca de 71 KB.
  3. Em cada computador cliente executar install\instalar.exe a partir da directoria que foi criada no servidor no passo 1. Se a rede for rápida, pode-se escolher a instalação "Compacta". Caso contrário, será preferível instalar parte do ToonTalk nos clientes, escolhendo a instalação "Típica" ou "Personalizada".
  4. No final podem apagar-se todas as directorias do Servidor, excepto a directoria ToonTalk(140 MB, pelo menos).

Caso se pretenda que a directoria dos utilizadores também fique no servidor, será necessário editar o ficheiro toontalk.ini na directoria do Windows de cada computador cliente. Na secção "Directories" mudar em  "MainDir" para a directoria no servidor. Deve ficar igual à directoria em "CDROMdir".
