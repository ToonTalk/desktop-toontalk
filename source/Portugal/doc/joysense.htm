Sensores do Joystick no ToonTalk

Se tiveres um joystick ligado ao teu computador, deverás encontrar um caderno de sensores para cada joystick que tiveres ligado. Aqui está a descrição dos sensores:

  • Página 2. Velocidade X do Joystick. Este sensor mostra a distância a que o joystick foi movido para a esquerda ou para a direita. Varia de -1.000 a 1.000.
  • Página 4. Velocidade Y do Joystick. Este sensor mostra a distância a que o joystick foi movido para a frente ou para trás. Varia de -1.000 a 1.000.
  • Página 6. Velocidade Z do Joystick. Este sensor mostra a distância a que o joystick foi movido para cima ou para baixo. Se não existir esta funcionalidade, o valor será zero. Varia de -1.000 a 1.000.
  • Página 8. Eixo X do Joystick. Rotação para a esquerda ou para a direita . Se não existir esta funcionalidade, o valor será zero. Varia de -1.000 a 1.000.
  • Página 10. Eixo Y do Joystick. Rotação para a frente ou para trás . Se não existir esta funcionalidade, o valor será zero. Varia de -1.000 a 1.000..
  • Página 12. Eixo Z do Joystick. Rotação para cima ou para baixo . Se não existir esta funcionalidade, o valor será zero. Varia de -1.000 a 1.000.
  • Página 14. Botões do Joystick Clicados. ToonTalk pode reconhecer um máximo de 32 botões. Quando um botão é clicado o seu valor é temporariamente "sim". Caso contrário é "não".
  • Página 16. Botões do Joystick pressionados. Isto mostra "sim" se o botão correspondente do joystick está pressionado, caso contrário mostra "não".

Se tiveres um  joystick force feedback, também existirá um caderno especial de sensores para controlar os efeitos de força.
