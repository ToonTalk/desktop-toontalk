Do you want to change which language ToonTalk speaks? For ToonTalk use by WebLabs in Sweden and Portugal (and other countries when ToonTalk is translated), I recommend you try using English as the spoken language. This is due the lack of an adequate text-to-speech engine in Swedish or European Portuguese. If you have already chosen English as the display language then English is already the default spoken language as well.

If you want to customize ToonTalk in other ways, then click on 'Set Options' in the ToonTalk start screen.
